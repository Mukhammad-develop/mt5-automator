# Changelog

All notable changes to the MT5 Trading Automator project will be documented in this file.

## [0.1.0] - 2025-12-04

### Added
- Initial release
- Telegram signal monitoring with Telethon
- OCR image processing with advanced preprocessing
- Smart signal parser with validation
- MT5 trading engine with automatic order placement
- Risk-based lot calculation
- Position tracking with breakeven logic
- TP2 protection system
- Comprehensive logging system
- Docker deployment support
- Complete documentation and tests

### Features
- Text and image signal support
- Multiple TP/SL levels
- Configurable position count (3, 6, 9+)
- Automatic reconnection
- Error recovery
- Real-time monitoring

### Security
- Environment variable-based credentials
- No hardcoded passwords
- Session management for Telegram
