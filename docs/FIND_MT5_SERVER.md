# Как найти MT5_SERVER (Имя сервера MT5)

Вам нужно найти имя сервера вашего брокера для настройки `MT5_SERVER` в файле `config.env`.

## Способ 1: Через меню MT5 (Самый простой)

1. **Откройте MetaTrader 5**
2. **Нажмите на меню:** `Сервис` (Tools) → `Настройки` (Options)
   - Или нажмите `Ctrl + O`
3. **Перейдите на вкладку:** `Сервер` (Server)
4. **Посмотрите на поле "Сервер"** - там будет указано имя сервера
   - Например: `MetaQuotes-Demo`, `ICMarkets-Demo`, `VantageInternational-Demo`

## Способ 2: Через информацию о счёте

1. **В MT5 нажмите:** `Сервис` (Tools) → `Свойства счёта` (Account Properties)
   - Или нажмите `Ctrl + I`
2. **В открывшемся окне найдите поле "Сервер"**
   - Там будет указано имя сервера

## Способ 3: Через окно входа (Login)

1. **Выйдите из MT5** (если вы вошли)
2. **При входе в MT5** в окне авторизации:
   - В выпадающем списке "Сервер" будет показано имя сервера
   - Это то, что вам нужно!

## Способ 4: Через журнал (Journal)

1. **В MT5 откройте вкладку "Терминал"** (Terminal) внизу
2. **Перейдите на вкладку "Журнал"** (Journal)
3. **Посмотрите записи при запуске MT5:**
   - Там может быть указано имя сервера в логах подключения

## Способ 5: Через навигатор (Navigator)

1. **В левой панели MT5 найдите "Навигатор"** (Navigator)
2. **Раскройте "Счета"** (Accounts)
3. **Найдите ваш счёт** - рядом с номером счёта может быть указан сервер

## Примеры имён серверов

Имена серверов обычно выглядят так:
- `MetaQuotes-Demo` - демо-счёт MetaQuotes
- `ICMarkets-Demo` - демо-счёт IC Markets
- `VantageInternational-Demo` - демо-счёт Vantage International
- `YourBroker-Demo` - демо-счёт вашего брокера
- `YourBroker-Live` - реальный счёт вашего брокера

**Важно:** 
- Для демо-счёта обычно добавляется `-Demo` в конце
- Для реального счёта может быть `-Live`, `-Real` или без суффикса
- Имя сервера **чувствительно к регистру** - пишите точно так, как в MT5

## Как использовать в config.env

После того, как найдёте имя сервера, добавьте его в `config.env`:

```env
MT5_LOGIN=12345678
MT5_PASSWORD=YourPassword123
MT5_SERVER=VantageInternational-Demo
```

**Примеры:**
```env
# Если сервер называется "ICMarkets-Demo"
MT5_SERVER=ICMarkets-Demo

# Если сервер называется "MetaQuotes-Demo"
MT5_SERVER=MetaQuotes-Demo

# Если сервер называется "YourBroker-Demo"
MT5_SERVER=YourBroker-Demo
```

## Проверка

После настройки проверьте подключение:

```powershell
python test_system.py
```

Если MT5 подключение проходит успешно, значит сервер указан правильно!

## Если не можете найти

1. **Свяжитесь с вашим брокером** - они точно знают имя сервера
2. **Посмотрите в документации брокера** - обычно там указано имя сервера для MT5
3. **Проверьте email от брокера** - при регистрации счёта обычно присылают данные для подключения

## Важные замечания

- ⚠️ **Имя сервера чувствительно к регистру** - пишите точно так, как в MT5
- ⚠️ **Не добавляйте пробелы** в начале или конце
- ⚠️ **Для демо и реальных счётов** могут быть разные имена серверов
- ✅ **Обычно имя сервера** содержит название брокера и тип счёта (Demo/Live)
