# Установка Python на Windows - Русская версия

Простая пошаговая инструкция для установки Python на Windows.

## Быстрая установка (5 минут)

### Шаг 1: Скачать Python

1. Откройте браузер (Chrome, Edge, Firefox)
2. Перейдите на сайт: **https://www.python.org/downloads/**
3. Нажмите большую жёлтую кнопку **"Download Python 3.x.x"**
   - Это самая новая версия (например, 3.12.0)
   - **Важно:** Нужна версия 3.9 или выше

### Шаг 2: Установить Python

1. **Найдите скачанный файл** в папке "Загрузки" (Downloads)
   - Файл будет называться примерно так: `python-3.12.0-amd64.exe`
2. **Дважды кликните** на файл, чтобы запустить установку
3. **ВНИМАНИЕ!** В самом низу окна установки есть галочка:
   - ✅ **"Add Python to PATH"** (Добавить Python в PATH)
   - **ОБЯЗАТЕЛЬНО отметьте эту галочку!** Без неё Python не будет работать из командной строки
4. Нажмите **"Install Now"** (Установить сейчас)
5. Подождите, пока установка завершится (1-2 минуты)
6. Когда увидите "Setup was successful", нажмите **"Close"** (Закрыть)

### Шаг 3: Проверить, что всё работает

1. Нажмите `Win + X` и выберите **"Windows PowerShell"** или **"Терминал"**
2. Введите команду:
   ```powershell
   python --version
   ```
3. Должно показать что-то вроде: `Python 3.12.0`

**Если показывает версию - отлично! Python установлен! ✅**

**Если показывает ошибку** - см. раздел "Проблемы" ниже.

## Альтернативный способ: Microsoft Store

Если не хотите скачивать с сайта:

1. Откройте **Microsoft Store** (Магазин Microsoft)
2. В поиске введите: **"Python 3.12"**
3. Выберите **"Python 3.12"** от Python Software Foundation
4. Нажмите **"Получить"** или **"Установить"**
5. Дождитесь установки

**Плюс этого способа:** Автоматически добавляет Python в PATH.

## Проверка установки

После установки проверьте эти команды в PowerShell:

```powershell
# 1. Проверить версию Python
python --version
# Должно показать: Python 3.12.x

# 2. Проверить pip (менеджер пакетов)
pip --version
# Должно показать: pip 24.x.x

# 3. Проверить, что Python работает
python -c "print('Привет, Python!')"
# Должно показать: Привет, Python!
```

Если все три команды работают - Python установлен правильно! ✅

## Устранение проблем

### Проблема: "python не распознан как команда"

Это означает, что Python не добавлен в PATH.

**Решение 1: Переустановить с PATH (Проще всего)**
1. Откройте "Параметры" → "Приложения" → "Установленные приложения"
2. Найдите "Python 3.x" и нажмите "Удалить"
3. Скачайте Python заново с https://www.python.org/downloads/
4. При установке **ОБЯЗАТЕЛЬНО отметьте "Add Python to PATH"**
5. Перезапустите PowerShell

**Решение 2: Использовать py launcher**
Windows обычно устанавливает `py` launcher, попробуйте:
```powershell
py --version
py -m pip --version
```

Если это работает, используйте `py` вместо `python`:
```powershell
py setup.py
py check_setup.py
```

**Решение 3: Добавить Python в PATH вручную**
1. Найдите, где установлен Python:
   - Обычно: `C:\Users\ВашеИмя\AppData\Local\Programs\Python\Python312\`
   - Или: `C:\Program Files\Python312\`
   - Или: `C:\Python312\`
   
2. Чтобы найти точно:
   - Откройте Проводник
   - В адресной строке введите: `%LocalAppData%\Programs\Python`
   - Или поищите папку "Python" в `C:\Program Files\`

3. Добавьте в PATH:
   - Нажмите `Win + R`
   - Введите `sysdm.cpl` и нажмите Enter
   - Нажмите "Переменные среды" (Environment Variables)
   - В разделе "Системные переменные" найдите `Path`
   - Нажмите "Изменить" (Edit)
   - Нажмите "Создать" (New) и добавьте путь к Python:
     - Например: `C:\Users\ВашеИмя\AppData\Local\Programs\Python\Python312`
   - Нажмите "Создать" ещё раз и добавьте путь к Scripts:
     - Например: `C:\Users\ВашеИмя\AppData\Local\Programs\Python\Python312\Scripts`
   - Нажмите "OK" на всех окнах

4. **Перезапустите PowerShell** (закройте и откройте заново)

5. Проверьте: `python --version`

### Проблема: Несколько версий Python

Если у вас установлено несколько версий Python:

```powershell
# Использовать конкретную версию
py -3.12 --version
py -3.11 --version

# Или указать полный путь
C:\Python312\python.exe --version
```
